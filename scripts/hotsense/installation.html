Installation

Installing and configuring HotSense is fairly easy for most webmasters.
I'll explain step-by-step:

Step 1

Download the package from the download folder and extract it in a local folder.

Step 2

Open this file in a text editor: .htaccess

RewriteEngine On
RewriteCond %{HTTP_REFERER} !^http://(\www\.)?yourdomain\.com/ [NC]
RewriteCond %{HTTP_REFERER} !^$
RewriteCond %{REQUEST_FILENAME} .*jpeg$|.*jpg$|.*JPEG$|.*JPG$ [NC]
RewriteRule .*\.(gif|jpg|jpeg|bmp)$ http://www.yourdomain.com/hotsense/hotlink.php?site=http://www.yourdomain.com&img=%{REQUEST_URI} [R,NC]

I've made everything you should edit bold.

Second row: The domain which has access to your images, you can and more by copying the line filling in another domain.
Fifth row: In both cases, it's the domain that will be checked by the script. So it's the same domain as in row 2 (most of the times).

Step 3

It's time to install the database table.
Open database_table.sql, copy the text, and execute the sql on the desired database. You can execute SQL with phpMyAdmin, which is most likely installed on your server.

Step 4

Time to change the configure file!

The first block is to configure your database connection settings.
You can set the image to be shown after a delay at the bottom and delay etc.

Step 5

Upload the configured files of the local folder to your root on your webserver. It's the www of httpdocs folder in most cases.

Step 6

You are finished!
You can check statistics on the www.yourdomain.com/hotsense/checker.php page!
