ComicSense is a script using php / mySQL. It allows you to easily host an Online Comic or Image shack.
Reasons to use ComicSense:
  • Easy to implent
  • Works fast
  • Comment system
  • News board
  • Archive System
  • Records stats
  • Protected admin area
  • Easy to adjust CSS
  • xhtml strict valid!

How To Install
  1. Download the latest version of ComicSense

  2. Unpack and upload to the location you desire

  3. Configure the 'includes/connect.php' file with textpad to your MySQL preferences

  4. Open the website and open install.php

  5. Run the setup

  6. Delete install.php

  7. Start having fun!
